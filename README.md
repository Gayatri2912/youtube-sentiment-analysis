# 🎯 YouTube Sentiment Analysis – End-to-End MLOps Pipeline

An end-to-end YouTube comment sentiment classification system built using Machine Learning and modern MLOps practices. The project demonstrates a complete workflow from data preprocessing and model experimentation to versioning, CI/CD automation, containerization, and cloud deployment.

---

## 🚀 Project Overview

This system analyzes YouTube comments and classifies them into sentiment categories (Positive / Negative / Neutral).  

It showcases:
- Structured ML pipeline development
- Experiment tracking and reproducibility
- Model and data versioning
- Automated CI/CD workflow
- Cloud deployment on AWS

---

## 🛠️ Tech Stack

**Languages & Libraries:**  
Python, Pandas, NumPy, scikit-learn, LightGBM, TF-IDF, imbalanced-learn (SMOTE)

**MLOps Tools:**  
MLflow (experiment tracking), DVC (data & model versioning), Docker (containerization), GitHub Actions (CI/CD)

**Cloud Platform:**  
AWS EC2 (deployment), AWS S3 (artifact storage)

---

## 📊 ML Workflow

- Performed data cleaning, preprocessing, and exploratory data analysis (EDA)
- Applied TF-IDF vectorization for feature engineering
- Handled class imbalance using SMOTE
- Experimented with multiple models (LightGBM, Random Forest, XGBoost)
- Selected best-performing model based on evaluation metrics
- Logged parameters, metrics, and artifacts using MLflow
- Version-controlled datasets and models using DVC

---

## ⚙️ Deployment & Automation

- Containerized the application using Docker
- Implemented CI/CD pipeline using GitHub Actions
- Deployed the model on AWS EC2
- Stored artifacts securely on AWS S3

---

## 📌 Key Highlights

- Production-ready MLOps pipeline
- Reproducible and version-controlled workflow
- Automated training-to-deployment lifecycle
- Scalable cloud-based deployment architecture
